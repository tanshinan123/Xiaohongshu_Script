项目描述：
小红书内容创作助手是一款面向内容平台的AIGC写作工具。系统调用GPT系列及头部模型，结合平台内容风格与话题语境，自动生成多候选标题与规范化正文，并通过结构化解析保证输出质量与一致性。适用于品牌营销、产品推广与个人IP内容创作，显著提升创作效率与内容质量。

项目概述：
专门针对小红书平台的内容创作工具，基于LangChain和Pydantic输出解析器，能够根据用户主题自动生成符合小红书平台特色的标题和正文内容。

技术架构：
- 前端框架：Streamlit - 提供简洁的创作界面
- AI框架：LangChain - 构建内容生成链
- 输出解析：PydanticOutputParser - 确保结构化输出
- 数据模型：Pydantic BaseModel - 定义输出数据结构
- 大语言模型：GPT-3.5-turbo - 内容生成引擎

核心功能：
1. 多标题生成：一次性生成5个不同风格的标题选项
2. 正文创作：生成符合小红书平台特色的正文内容
3. 结构化输出：通过Pydantic模型确保输出格式的一致性
4. 平台适配：针对小红书用户习惯和内容风格进行优化
5. 批量生成：支持一次生成多个标题，提高创作效率

技术亮点：
- 使用PydanticOutputParser实现严格的结构化输出控制
- 自定义Pydantic模型定义输出格式，确保数据质量
- 针对特定平台的内容风格优化
- 简洁的用户界面，降低使用门槛
- 支持批量标题生成，提升创作效率

应用场景：
- 小红书内容创作
- 社交媒体营销
- 生活方式分享
- 产品推广文案
- 个人品牌建设

项目价值：
展示了AIGC技术在垂直平台内容创作中的应用，体现了对特定用户群体和平台特性的深度理解，通过技术手段解决了内容创作者在标题和正文创作方面的痛点，为社交媒体内容生产提供了专业化的解决方案。
