项目描述：
小红书内容创作助手是一款面向内容平台的AIGC写作工具。系统调用GPT系列及头部模型，结合平台内容风格与话题语境，自动生成多候选标题与规范化正文，并通过结构化解析保证输出质量与一致性。适用于品牌营销、产品推广与个人IP内容创作，显著提升创作效率与内容质量。

项目概述：
专门针对小红书平台的内容创作工具，基于LangChain和Pydantic输出解析器，能够根据用户主题自动生成符合小红书平台特色的标题和正文内容。

技术架构：
- 前端框架：Streamlit - 提供简洁的创作界面
- AI框架：LangChain - 构建内容生成链
- 输出解析：PydanticOutputParser - 确保结构化输出
- 数据模型：Pydantic BaseModel - 定义输出数据结构
- 大语言模型：GPT-3.5-turbo - 内容生成引擎

核心功能：
- 多标题生成：一次性生成5个不同风格的标题选项
- 正文创作：生成符合小红书平台特色的正文内容
- 结构化输出：通过Pydantic模型确保输出格式的一致性
- 平台适配：针对小红书用户习惯和内容风格进行优化
- 批量生成：支持一次生成多个标题，提高创作效率

技术亮点：
- 使用PydanticOutputParser实现严格的结构化输出控制
- 自定义Pydantic模型定义输出格式，确保数据质量
- 针对特定平台的内容风格优化
- 简洁的用户界面，降低使用门槛
- 支持批量标题生成，提升创作效率

应用场景：
- 小红书内容创作
- 社交媒体营销
- 生活方式分享
- 产品推广文案
- 个人品牌建设

项目价值：
展示了AIGC技术在垂直平台内容创作中的应用，体现了对特定用户群体和平台特性的深度理解，通过技术手段解决了内容创作者在标题和正文创作方面的痛点，为社交媒体内容生产提供了初步的解决方案。

Video_Script_Generator - AIGC项目描述

项目描述：
视频生成器是一款基于最新人工智能生成内容（AIGC）技术的创新工具。该工具通过调用GPT系列及头部模型，根据用户输入的主题调用维基百科等API联网查询最新相关信息，作为提示词的上下文语境。系统将结合视频时长及词义随机性要求输出合适的脚本内容。适用于内容创作者、企业宣传等多种应用场景。

项目概述：
基于LangChain和维基百科API的短视频文案生成工具，能够根据用户输入的主题、时长和创意度参数，自动生成吸引人的短视频标题和结构化脚本内容。

技术架构：
- 前端框架：Streamlit - 提供参数配置和结果展示界面
- AI框架：LangChain - 构建多步骤生成链
- 外部知识：WikipediaAPIWrapper - 获取主题相关信息
- 提示工程：ChatPromptTemplate - 结构化提示词管理
- 大语言模型：GPT-4-turbo - 内容生成引擎

核心功能：
1. 智能标题生成：基于主题生成吸引流量的短视频标题
2. 结构化脚本：按照【开头】-【中间】-【结尾】的格式生成完整脚本
3. 时长适配：根据用户指定的视频时长调整内容长度
4. 创意度控制：通过temperature参数控制内容的创新程度
5. 知识增强：结合维基百科信息提升内容准确性
6. 风格定制：针对95后上班族群体优化内容风格

技术亮点：
- 采用多步骤生成链，先生成标题再生成脚本
- 集成外部知识源，提升内容质量和准确性
- 可调节的创意度参数，满足不同创作需求
- 结构化的输出格式，便于后续视频制作
- 异常处理机制，确保系统稳定性

应用场景：
- 短视频内容创作
- 社交媒体营销
- 教育培训视频
- 产品宣传片
- 个人IP打造

项目价值：
展示了AIGC在内容创作领域的强大能力，通过技术手段降低了短视频创作的门槛，体现了对内容创作流程的深度理解和产品化思维，为内容创作者提供了高效的生产工具。
